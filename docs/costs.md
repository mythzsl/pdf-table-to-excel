# Cost Plan

## Low-cost validation budget

| Item | Expected cost |
| --- | ---: |
| `.com` domain | $10-15/year |
| Cloudflare Pages frontend | $0/month |
| Railway backend | $5-25/month early usage |
| Analytics | $0/month |
| Email/waitlist form | $0-15/month |
| Ahrefs Starter during research months | $29/month optional |

Expected v1 monthly cost: **$35-80/month**, excluding the annual domain fee.

## OCR later

OCR is intentionally excluded from v1 default conversion.

Indicative AWS Textract costs:

- Detect Document Text OCR: about $0.0015/page
- Analyze Document Tables: about $0.015/page
- 1,000 table pages/month: about $15
- 10,000 table pages/month: about $150

Only launch OCR after adding quotas, account-based billing, or a paid beta.
